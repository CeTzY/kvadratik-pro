# Kvadratik Pro

Пиксельный редактор с анимацией и слоями.

## Возможности
- 5 видов кистей (круг, текстура, спрей, звезда)
- Анимация и экспорт в GIF
- Слои с прозрачностью
- Фильтры и эффекты
- Горячие клавиши
